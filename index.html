Octagon

Coffeescript, Javascript, Redis

Octagon is a small and efficient backend for creating ticketing systems. It it written mostly in Coffeescript with hints of Javascript and the data is effectively stored in Redis. All of the methods are created with Promises in mind making them easy to integrate in your existing code. I chose to use Coffeescript and Redis here for both efficiency, practice and readability.

Teachery

Javascript, CSS, PHP

Teachery is a web application tool designed to help teachers out with common taks under which sorting and performing asynchronous countdowns. It works on both mobile and web and has unofficial inegration with Magister, which is a popular school management system in there Netherlands. We decided to create web application with these techniques to create as support as possible.

Lytebot

Python, API

A small bot I started working on with 2 other students. It is mainly designed to show us pictures of cats but can also search on the populair search engine duckduckgo, use their convenient bangs, Scrape subreddits for relevant images and has a small admin system implemented which can disable commands and have the bot ignore users. We went with Python for familiarity in this scenario.

Moe

Bash, cURL, Groff

Moe is an tool created to upload files quickly. It was originally created to upload screenshots quickly to a single site, but that time has past (un)fortunately. It now uploads to any site that relies on pomf.se source code. I used bash for native usage of the cURL functionality.
